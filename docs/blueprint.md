# **App Name**: BloomVision

## Core Features:

- Define Success Carousel: Carousel to input and display success definitions with a motivational banner.
- Life Vision Accordion: Accordion form to input life vision milestones for decades and 5-year plans.
- Action Plan Generator: Checklist builder to create and manage action steps towards achieving life vision, with plan summary generation.
- Vision Statement Builder: Input field for vision statement and measurable 12-month goal selector.
- Vision Board: Drag-and-drop interface for uploading and arranging images to create a vision board with caption support.
- AI Progress Reflection Tool: AI powered tool that analyzes journal entries and vision board elements to offer daily progress reflections. The tool decides whether to incorporate information on vision elements based on user activity and patterns.
- Travel Map: Interactive world map with a color-coded pin system for marking visited destinations and planned travels.

## Style Guidelines:

- Primary color: Soft rose gold (#B76E79) to evoke sophistication and warmth.
- Background color: Light beige (#F5F5DC) for a calm and modern aesthetic.
- Accent color: Dusty lavender (#D8BFD8) to highlight important actions and guide the user.
- Header font: 'Literata', a soft serif, for titles and prominent text.
- Content font: 'Inter', a sans-serif, for body text and other content. 
- Minimal, rounded icons for a soft and approachable user interface.
- Use of white space to create a calm and uncluttered design.
